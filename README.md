# Rempah Nusantara

Website profil perusahaan dan katalog produk untuk **Sekawan Cipta Dinar**, eksportir rempah-rempah berkualitas tinggi dari Indonesia ke pasar global.

## Fitur Utama

- **Multibahasa**: Mendukung Bahasa Indonesia dan Bahasa Inggris.
- **Katalog Produk**: Daftar produk rempah dengan detail lengkap (segar, kering, bubuk).
- **Integrasi WhatsApp**: 
  - Tombol melayang WhatsApp untuk respon cepat.
  - Formulir pemesanan yang langsung mengirim pesan ke WhatsApp dengan detail nama, email, dan pesan pembeli.
- **Katalog PDF**: Fitur cetak/simpan katalog produk dan profil perusahaan ke format PDF secara rapi.
- **Optimasi Performa**: Kode CSS dan JavaScript yang bersih, minimalis, dan efisien.
- **Responsif**: Tampilan yang menyesuaikan dengan berbagai perangkat (desktop, tablet, mobile).
- **Favicon**: Ikon situs yang konsisten di semua halaman.

## Struktur Proyek

- `index.html`: Halaman utama profil perusahaan.
- `katalog-produk.html`: Halaman daftar lengkap produk.
- `katalog-company.html`: Halaman profil detail perusahaan.
- `detail-produk.html`: Halaman detail untuk masing-masing item produk.
- `css/`: Direktori file gaya (style.css, detail-produk.css).
- `script/`: Direktori logika aplikasi JavaScript.
- `data/`: Penyimpanan data produk (`products.json`) dan terjemahan (`translations.json`).
- `image/`: Direktori aset gambar dan favicon.

## Teknologi yang Digunakan

- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
- Google Fonts (Poppins & Playfair Display)

## Cara Menggunakan

1. Buka `index.html` di browser pilihan Anda.
2. Gunakan switcher bahasa di pojok kanan atas untuk mengganti bahasa.
3. Klik tombol WhatsApp melayang atau isi formulir di bagian bawah untuk menghubungi admin.
4. Di halaman katalog, gunakan tombol "Download Katalog" untuk menyimpan profil atau daftar produk sebagai PDF.
